# Employee Performance Analysis Dashboard

## Project Overview

This project was created using Microsoft Excel to analyze employee performance data through an interactive dashboard.

## Tools Used

* Microsoft Excel
* Pivot Tables
* KPI Cards
* Slicers

## Features

* Interactive Dashboard
* Employee Performance Analysis
* KPI Tracking
* Dynamic Filtering using Slicers
* Data Summarization using Pivot Tables
